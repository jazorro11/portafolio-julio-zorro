# Julio Zorro — Portfolio

> I build intelligent systems and the interfaces that make them feel human.

Personal portfolio site showcasing work at the intersection of software engineering and AI.

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | Next.js 16 + React 19 (App Router) |
| Language | TypeScript 5 |
| Styling | Tailwind CSS v4 |
| Animations | Framer Motion 12 — entrance animations & scroll reveals |
| 3D / WebGL | Three.js — particle field in the Hero section |
| Smooth scroll | Lenis |
| i18n | next-intl — locale routes `/en` and `/es` |

## Getting Started

**Prerequisites:** Node.js 18+

```bash
git clone https://github.com/juliozorro/portafolio-julio-zorro.git
cd portafolio-julio-zorro
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev    # Development server with HMR
npm run build  # Production build
npm start      # Start production server
```

## Internationalization

The site is available in English and Spanish via locale-based routing:

- `/en` — English
- `/es` — Español

The default locale redirects automatically based on browser preference.
